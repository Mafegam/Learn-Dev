HTML Elements

HTML elements are the building blocks of HTML pages, representing different types of content such as headings, paragraphs, images, links, forms, and more.

An HTML element generally consists of the following parts: 

<tagname attribute ="value">Content</tagname>
  • Tag name: The name of the HTML element, such as <p> for paragraphs, <h1> for headings, <a> for links, etc.
  • Attributes: Optional attributes provide additional information about an element. Attributes are placed within the opening tag and can modify the element's behavior or appearance. Example: <href> attribute for links.
  • Content: The actual content enclosed between the opening and closing tags. This could be text, other HTML elements, or multimedia.

Commonly Used HTML Elements

  1. Headings: Used for titles and subtitles 

    <h1>This is a heading</h1>   
<h2>This is a subheading</h2>

  2. Paragraphs: Used for blocks of text. 

    <p>This is a paragraph.</p>

  3. Links: Used for navigation or directing users to other web pages. 

    <a href="https://www.example.com">Visit Example</a>

  4. Images: Used to display images on a web page. 

    <img src="image.jpg" alt="Description of the image">>

  5. Lists: Used to create ordered or unordered lists. 

    <ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>                   
                                
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>

  6. Forms: Used to collect user input. 

    <action="/submit-form">
    <input type="text" name="username" placeholder="Enter your username">
    <button type="submit">Submit</button>
</form>

Conclussion

Understanding HTML elements is essential for web development. By mastering these basics, you can create structured and meaningful web pages. Experiment with different elements and attributes to see how they affect the appearance and functionality of your web pages.
