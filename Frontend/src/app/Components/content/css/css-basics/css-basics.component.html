CSS Basics

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS controls the layout, colors, fonts, and other visual aspects of web pages. Let's dive into the basics of CSS:

Introduction to CSS

CSS works by selecting HTML elements and applying styles to them. Styles can include things like colors, fonts, margins, padding, and more.

Anatomy of a CSS Rule

A CSS rule consists of a selector and a declaration block:

selector{
property: value;
/*more properties and values*/
}
  • Selector: It targets HTML elements to which you want to apply styles. It can be an element selector (e.g., h1, p), class selectors (e.g., .container), ID selector (e.g., #header), or more complex selectors (e.g., nav, ul, a).
  • Declaration Block: OIt contains one or more declarations separated by semicolons (;). Each declaration includes a property (e.g., color, font-size) and its corresponding value (e.g., blue, 16px).

Using CSS

Internal CSS:You can include CSS within an HTML document using the <style> tag in the <head> section: 

<!DOCTYPE html>   
<html>
<head>
  <style>
    /* CSS styles go here */
    body {
        font-family: Arial, sans-serif;
        background-color: #f0f0f0l;
    }
    h1 {
        color: blue;
        text-align: center;
        }
  </style>
</head>
<body>
    <h1>Welcome to my website!</h1>
    <bpody>This is a paragraph</p>
</body>
</html>

External CSS:Alternatively, you can link to an external CSS file using the <link> tag in the <head> section: 

<!DOCTYPE html>   
<html>
<head>
  <link rel="stylesheet" type="text/css" href="styles.css">
</head>
<body>
    <h1>Welcome to my website!</h1>
    <bpody>This is a paragraph</p>
</body>
</html>

Conclussion

CSS is essential for styling and formatting web pages. By mastering these basics, you can create visually appealing and responsive layouts for your websites. Experiment with different selectors, properties, and values to see how they affect the appearance of HTML elements.
