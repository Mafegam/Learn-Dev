CSS Selectors

CSS selectors are patterns used to select and style HTML elements based on their tag name, class, ID, attributes, and relationships with other elements. Understanding CSS selectors is essential for effectively styling web pages.

Element Selector

Selects all instances of a specific HTML element. 

body {
    font-size: 16px;
    color: #333;
}

This example applies styles to all <p> (paragraph) elements, setting their font size to 16 pixels and text color to dark gray.

Class Selector

Selects elements with a specific class attribute. 

.highlight {
    background-color: yellow;
    font-weight: bold;
}

Apply this class to any element (<div class="highlight">...</div>) to give it a yellow background and bold text.

ID Selector

Selects a single element based on its unique ID attribute. 

#header {
    background-color: #333;
    color: white;
    padding: 10px;
}

Use #header to style a specific element with id="header, such as a header section.

Attribute Selector

Selects elements based on their attributes.

  • Selects all elements with a specific attribute: [type]
  • Selects elements with a specific attribute value: [type="text"]
  • Selects elements with an attribute starting with a specific value: [href^="https://"] 
input [type="text"] {
    width: 200px;
    padding: 5px;
}

This example styles all <input> elements with type="text", setting their width to 200 pixels and adding 5 pixels of padding.

Descendant Selector

Selects elements that are descendants of a specified element. 

ul li {
    list-style-type: square;
}

Styles all <li> (list items) that are descendants of <ul> (unordered list) elements, making them display with square bullets.

Child Selector

Selects direct child elements of a specified parent element. 

.nav > li {
    display: inline-block;
    margin-right: 10px;
}

Styles <li> elements that are direct children of elements with class .nav, making them display inline-block with 10 pixels of right margin.

Pseudo-classes and Pseudo-elements

Pseudo-classes are used to style elements when they are in a certain state or condition, like :hover, :focus, :first-child, etc. 

a:hover {
    color: red;
    text-decoration: underline;
}

p::first-line {
    font-weight: bold;
}

In this example, links (<a>) turn red and underlined when hovered over. The first line of paragraphs (<p>) is styled in bold.

Grouping Selectors

Allows you to apply the same styles to multiple selectors in one declaration. 

h1, p, li {
    color: blue;
}

This sets the text color of <h1>, <p>, and <li> elements to blue.

Conclussion

Understanding CSS selectors empowers you to precisely target and style elements within your HTML documents. By combining selectors intelligently, you can create visually appealing and responsive web designs.
