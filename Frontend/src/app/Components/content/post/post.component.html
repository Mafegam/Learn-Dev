Intro to HTML

HTML (HyperText Markup Language) is the standard language used to create and design web pages. It consists of a series of elements that you use to enclose, or wrap, different parts of the content to make it appear or act a certain way.

Every HTML document follows a basic structure:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
</body>
    <!-- Content goes here -->
</html>

Replace <!-- Content goes here--> with your actual content, such as headings, paragraphs, images, links, lists, forms, etc., depending on what you want to display on your webpage.

Here’s a simple example with some content:

<!DOCTYPE html>
    <html>
    <head>
        <title>My Website</title>
    </head>
    </body>
        <h1>Welcome to my website</h1>
        <p>This is a paragraph introducing my website</p>
        <img src="image.jpg" alt="Image description">
        <a href="about.html">Learn more about us>
    </html>
  • <!DOCTYPE html> Declares the document type and version of HTML (HTML5 in this case).
  • <html> Root element that wraps all content on the page.
  • <head> Container for meta-information about the document, such as title, links to CSS stylesheets, and scripts.
  • <title> Sets the title of the webpage (displayed in the browser tab).
  • <body> Contains all the content that will be displayed in the browser window.

In this example:

  • <h1> is used for the main heading.
  • <p> is used for a paragraph of text.
  • <img> displays an image.
