Formatting in HTML

HTML provides several tags and attributes for formatting text and content. These options help you structure and style your web pages effectively.

Tags to add format

HTML provides tags such as bold, italic, underline, and more.

  • Bold text: Use <strong> or <b> tags. 
    <strong>This text is bold</strong>   
<b>This text is also bold</b>
  • Italic text: Use <em> or <i>. 
    <em>This text is italicized</em>   
<i>This text is also italicized</i>
  • Underline: Use the <u> tag. 
    <u>This text is underlined</u>

Heading Tags

HTML heading tags (<h1> to <h6>) are used to define headings and subheadings on a web page. They range from largest (<h1>) to smallest (<h6>).

Line Breaks

Line break: Use <br1> to insert a line break. 

This text is on the first line.<br>   
This text is on the second line.

Quotations and Citations

  1. Blockquote: <li>blockquote</li> for longer quotations. 

    <blockquote>This is a longer quotation that spans multiple lines.</blockquote>

  2. Citation: Use <li>cite</li> tag to define the title of a work (e.g., a book, movie). 

    <p>The book <cite>The Great Gatsby</cite> was written by F. Scott Fitzgerald.</p>

Conclussion

HTML formatting tags allow you to enhance the readability and structure of your web content. By mastering these basic tags, you can create visually appealing and well-organized web pages.
